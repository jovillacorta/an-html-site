PIXEL ART office

a cozy 3d pixel art style office

DRAG AND SCROLL TO EXPLORE

I made this project in order to become more familiar with ThreeJS, a 3D Javascript library.

This project features a 3D office I built (and am still building) in Blender.

I added OrbitControls so that the user can look around the office.

This project is still in progress, as I'm still learning to model in Blender. I add the objects I model as I go.
