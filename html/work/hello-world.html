HELLO WORLD

a pixel art RPG created in UNITY

Hello world is a senior class team project, and one of my favorite projects to date. It was created in UNITY, with version control maintained via github.

Hello World a multi-level RPG pixel art game. Our player wakes up in a post apocalyptic world where rogue robots have taken over. The objective of the game is to fight these robots, and with each successful fight, the opposing robot will join your team.

Features include basic character movement, fight scenes, item inventory, character health and XP stats, story board, and save functions.

This project only got as far as one level due to the 10 week constraint of our class, but I would love to revisit this project and build more levels
