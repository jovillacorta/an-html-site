R-CHAT

an android messaging app

R-CHAT was a team project for a software development course. We used an agile-style scrum development process and built the project in Android Studio using Javascript.

Features include user accounts (creation and verification using Firebase), chat function with image/video file support, and group chat function.

The chat app is pretty basic, but I learned most from the team development process. It was great practice for team project planning and version control via github.
